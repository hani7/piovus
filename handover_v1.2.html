5. PROCÉDURE DE DÉPLOIEMENT

Frontend (React)

  1. Exécuter la compilation : cd frontend && npm run build
  2. Créer une archive ZIP du dossier dist/*
  3. Accéder au cPanel → File Manager et naviguer vers public_html/
  4. Vider l'ancien contenu, puis uploader et extraire la nouvelle archive ZIP

Backend (Django)

  1. Pousser les modifications sur Github : git add . && git commit -m "..." && git push
  2. Accéder au cPanel → Setup Python App
  3. Cliquer sur le bouton REDÉMARRER
  4. Le script passenger_wsgi.py s'occupe de faire un git reset --hard et de relancer l'application WSGI.
⚠️ Toujours cliquer sur "REDÉMARRER" dans cPanel (Setup Python App) après chaque push sur le repository backend, sinon les modifications ne seront pas prises en compte.
